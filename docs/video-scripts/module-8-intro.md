# Module 8 Intro — Power User Tools

- **Instructor:** Wes
- **Target runtime:** ~80 seconds
- **Audience:** Cashman employees, brand new to AI -- but now finishing up the course
- **Goal:** Frame Module 8 as "graduation," introduce Claude Cowork and Claude Code at a beginner-friendly level, and motivate the final assessment.

---

## On-screen title card

> **Module 8 — Power User Tools**
> Bigger problems. Better tools. And your personal AI playbook.

---

## Script

Hey -- Wes here, back to close out the course with Module 8.

If you've made it this far, take a second and appreciate that. You started with "what is AI?" and now you can fix emails, write reports, build spreadsheets, search company documents, and even run agents. That's real.

This last module is about leveling up. Two new tools, then a wrap-up.

The first is **Claude Cowork.** Think of it as the tool you reach for when the question is *big*. A 100-page contract you need to evaluate for risk. Three different project delivery methods you need to compare. A complicated proposal you need to stress-test before you send it. Claude Cowork is built for sustained, deep thinking -- the heavy lifting.

The second is **Claude Code.** This one's for the more technical folks. If you've ever wished a repetitive task could just *run itself* -- renaming a hundred files, parsing a CSV, building a quick utility -- that's what Claude Code does. If you're not technical, that's totally fine. You'll learn enough to know when to flag the opportunity to IT.

And then we close the course with the most important step of all: **building your personal AI workflow.** A simple decision tree. When you're staring at a task, which tool do you reach for? You'll have a clear answer.

We'll also recap every security risk you've learned -- prompt injection, hallucinations, deepfakes, agent over-trust, and the rest -- so it all clicks into place.

After this module, there's a final assessment. Pass it, and you've earned your **Think (AI)mpossible** badge and the Cashman AI Training certificate. You've got this. Let's finish strong.
