# Module Intro Video Scripts

Short intro scripts (one per module) for use in Descript. Each script is sized
for roughly **60–90 seconds** of voiceover and is written for someone who is
**brand new to AI** -- no jargon, no assumptions.

## How to use these in Descript

1. Open Descript and create a new project for the module.
2. Paste the **Script** section into a Descript Script Block (or use Overdub if
   you're cloning the instructor's voice).
3. Use the **B-roll / Visuals** notes to layer in screen recordings, stock
   footage, or images.
4. The **On-screen text** lines are designed to appear as lower-thirds or
   full-frame title cards.

## Style rules these scripts follow

- **Plain English.** No "LLM," no "transformer," no "embeddings." If a term
  needs to be introduced (like *hallucination* or *prompt*), it gets a
  one-sentence definition.
- **Short sentences.** Easier to read on a teleprompter, easier for the
  audience to follow.
- **Conversational pauses** are marked with line breaks -- each line is
  roughly one breath.
- **Cashman-specific framing.** Examples reference marine construction,
  project work, and the tools Cashman employees actually have access to.
- **Each video ends with a hand-off** into the first lesson so the viewer
  knows exactly what to do next.
